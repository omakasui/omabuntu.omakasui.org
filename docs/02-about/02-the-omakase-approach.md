# The Omakase Approach

Setting up a Linux desktop from scratch takes time. A lot of it. Choosing the right tools, configuring them to work well together, establishing a consistent visual identity, and wiring everything up with sensible shortcuts... this is hours, or days, of work before you can even think about being productive.

That's the problem Omakase-style setups like Omakub, Omarchy, and Omabuntu are designed to solve.

## One Command, Ready to Work

The entire system is set up with a single command. You get a configured editor, a terminal with multiplexing, a curated set of tools, a theme system, keybindings, extensions, and a control menu, all integrated and consistent with each other. You don't make dozens of small decisions; the setup makes them for you, based on tested, opinionated choices.

This is what _omakase_ means: chef's choice. You trust that the selection is good, and you get on with your work.

## Curated, Not Bloated

Omabuntu ships with a minimal but complete set of tools, chosen for quality and coherence. The pre-installed set is intentionally small; everything optional is one menu interaction away. And anything can be removed, bringing the system back to a lean base whenever needed.

## A Consistent Environment

Themes apply everywhere at once: terminal, editor, UI elements, even the boot and login screens. Hotkeys are designed to work across apps. The launcher, the menu, and the configuration scripts all share the same mental model. This consistency is not accidental; it's the direct result of having a single setup that owns the full environment.

## Stable and Predictable

Built on Ubuntu LTS, Omabuntu targets long-term support releases so the base never shifts under your feet. Security updates land promptly; the package ecosystem is stable; the behavior is reproducible. The kind of foundation that matters when your machine is a professional tool, not an experiment.

## Easy to Maintain and Extend

Because the configuration is centralized, managed, and scripted, updates to the whole system are one command away. Adding a new theme, switching a font, or installing a new tool happens from the same unified menu. The underlying structure makes it easy to customize without breaking the cohesion.
