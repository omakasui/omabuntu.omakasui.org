# Commercial Applications

Some of the optional applications available in the Omabuntu Menu are commercial or freemium. All are installable via _Install > App_.

## 1Password

[1Password](https://1password.com) is a password and secrets manager. The install also includes the `op` CLI for scripting vault access.

## Typora

[Typora](https://typora.io) is a minimal Markdown editor with a live-preview rendering mode. It comes preconfigured with the iA Writer theme in both light and dark variants (matching the active Omabuntu theme).

Typora offers a 15-day free trial; a license is a one-time purchase.

## Spotify

[Spotify](https://spotify.com) is the popular music streaming client.

## Dropbox

[Dropbox](https://dropbox.com) is a cloud file sync service.

## Bitwarden

[Bitwarden](https://bitwarden.com) is an open-source password manager with a free tier. It's an alternative to 1Password.

## Zoom

[Zoom](https://zoom.us) is a video conferencing client.

## Steam

[Steam](https://store.steampowered.com) is the PC gaming platform by Valve.
