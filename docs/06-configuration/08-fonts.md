# Fonts

Omabuntu uses _Cascadia Mono Nerd Font_ as the default font for the terminal, while the system interface keeps _Ubuntu's default fonts_.

If you want to change the terminal font, open the _Omabuntu Menu_ with `Super + Alt + Space`, then go to _Style > Font_ and select a different option. You can also adjust the font size from _Style > Font > Size_.

You can install additional popular programming fonts from _Install > Style > Font_ in the same menu. If you no longer need an installed font, remove it via _Remove > Style > Font_.
